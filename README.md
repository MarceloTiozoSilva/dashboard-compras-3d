# Dashboard de Compras 3D

Dashboard interativo com gráficos 3D para análise de dados de compras industriais (2020–2025).

## Sobre o projeto

Desenvolvido com Claude Code a partir de dados reais de notas fiscais, este dashboard visualiza 5 anos de histórico de compras de uma empresa industrial com foco em análise de fornecedores, impostos e sazonalidade.

## Stack

- [Plotly.js](https://plotly.com/javascript/) — gráficos 3D interativos
- Python + xlrd — extração e processamento dos dados `.xls`
- Claude Code — geração e execução do script
